## :information_source: How To Use

```bash
# Clone this repository
$ git clone https://github.com/eltonlazzarin/reactjs-widget-trampos.git

# Go into the repository
$ cd reactjs-widget-trampos

# Install dependencies
$ yarn

# Run the app
$ yarn start
```
